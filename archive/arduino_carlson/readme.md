# Carlson: Chute Deployment and Logging System

This directory contains code for the first Carlson system.

## Wiring for Arduino Pro Micro board

### MPU6050 wiring:
    VCC, GND
    SCL  <->  3
    SDA  <->  2

### SD card logger wiring:
    VCC, GND
    CS    <->  10
    SCK   <->  15
    MOSI  <->  16
    MISO  <->  14
